# Contact Management App

## Description
A full-stack MERN (MongoDB, Express, React, Node.js) application to manage contacts with authentication.

---

## Features
- User Authentication (Login and Signup)
- Add Contact
- Edit Contact
- Delete Contact
- Search Contacts
- Filter Contacts by Status (Interested, Follow-up, Closed)

---

## Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
